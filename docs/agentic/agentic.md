# Agentic

:::info Work in Progress
This section is under active development. The topics below outline what's coming —
content and code samples will be added as the work lands.
:::

This section covers how AI agents interact with Multipaz for **agentic identity and
commerce** — letting autonomous agents discover capabilities, present credentials, and
complete payments on a user's behalf.

A reference implementation built around an **MCP app** ties these pieces together,
combining the Model Context Protocol, x402 payments, and HTTP header signaling.

## Planned topics

- **MCP** *(coming soon)* — A Model Context Protocol app that lets AI agents request
  and verify digital credentials through Multipaz.
- **x402** *(coming soon)* — Agent-initiated payments using the x402 protocol.
- **Non-Human Present (NHP) flow** *(future)* — Presentation flows where no human is
  present to approve the interaction.
